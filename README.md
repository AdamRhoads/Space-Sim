How It Works

The simulator uses a simplified form of Newton’s Law of Universal Gravitation:

        m₁ · m₂
F = G · ——————
          r²

To keep things simple (and the numbers clean), I just set

G = 1

instead of using the real constant
6.673 × 10⁻¹¹ N·m²/kg²
